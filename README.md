# TaskNotes

Simple Android task app built with Jetpack Compose (Material 3).

## Features
- Task list with filters (all / pending / done)
- Create and edit tasks
- Mark tasks as done / pending
- Delete with confirmation dialog
- Persistence with DataStore + Gson (JSON)

## Tech
- Kotlin + Jetpack Compose
- Navigation Compose
- DataStore Preferences
- Gson
